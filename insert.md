# `insert()`

Inserts an empty row into the quadrille.

# Example

# Syntax

> `insert(row)`

# Parameters

| parameter | description                                                                     |
|-----------|---------------------------------------------------------------------------------|
| row       | Number: number of the row to be inserted [\[0..height\]]({{< ref "height" >}})] |
