# Mutable methods

The mutable methods modified the calling `quadrille`.

{{< hint info >}}
**Observation**\
The methods of this section may be [chained](https://en.wikipedia.org/wiki/Method_chaining), e.g., `quadrille.clear().fill(5, 'ğŸ').randomize()` which is equivalent to:
```js
quadrille.clear();
quadrille.fill(5, 'ğŸ');
quadrille.randomize();
```
{{< /hint >}}
